# Copywriting frameworks (pick one per piece)

Match the framework to the goal and format. Always render in the brand voice
from `revamio-context.md` and ICP verbatim language.

- **PAS** (Problem → Agitate → Solve) — landing pages, ads, cold email. Lead
  with the ICP's pain in their own words.
- **AIDA** (Attention → Interest → Desire → Action) — general marketing copy.
- **BAB** (Before → After → Bridge) — transformation/outcome stories.
- **FAB** (Feature → Advantage → Benefit) — product pages; always end on the
  benefit to the ICP, not the feature.
- **PASTOR** (Problem, Amplify, Story, Transformation, Offer, Response) —
  long-form sales pages.
- **SCQA** (Situation, Complication, Question, Answer) — thought leadership,
  exec posts.

## Format defaults
- **Blog/article:** hook → problem (ICP language) → insight/answer-first →
  proof → generalizable takeaway → soft CTA. 800–1500 words unless deep.
- **Landing:** PAS hero → 3 value props (benefit-led) → proof → single CTA.
- **LinkedIn:** one idea, short lines, a concrete story, no hashtag spam.
- **X:** one sharp claim; thread only if there's a real sequence.

## Always
- Quote ICP pain language verbatim; avoid vendor-speak.
- One real proof point minimum.
- Respect the never-use vocabulary list.
